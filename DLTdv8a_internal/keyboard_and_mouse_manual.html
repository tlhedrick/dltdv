List of DLTdv8 app mouse and keyboard commands:


Mouse commands:

In the video windows the following mouse commands are active:

  • Left click: Digitize the 2D location or select a point by clicking on a line

  • Right click: Remove digitized point (Ctrl + click in MacOS)

  • Middle click or scroll wheel click: Initiate multi-track (see also M)

  • Scroll wheel rotation: Zoom in or out


Keyboard commands:

These are typically active in the video and time-series windows:

  • f, b, F and B: move (f)orward or (b)ackward by 1 frame in the video; F & B move by “Big Step Size” frames

  • -, =, r: zoom in (-), zoom out (=) or (r)eset the video view

  • space, z: the space bar digitizes a point (equivalent to a mouse left-click) and z deletes it (same as a mouse right click)

  • i, j, k, m and 4, 8, 6, 2: nudge the current point in the current video by 0.5 pixels in any of 4 directions

  • x, X: x cycles through the Autotrack modes and mode menu, excluding multi-track. X jumps to multi-track mode. Using x with autotrack running will stop it immediately.

  • n, D: n creates a new point, D deletes the current point (with dialogue box confirmation)

  • . and ,: cycle the active point forward (.) and backward (,) through the list

  • < and >: jump to the beginning or end of the digitized data in the current point in the current video

  • J: merge two points by combining their 2D data where they do not overlap, averaging it where they do and then recomputing the 3D result if in 3D mode.

  • Y: move the data from 1 point in 1 camera into a new 2nd point over a specified frame range

  • U: undo the last point Delete, Merge or Move (i.e. D, J, Y) operation or Neural Network application from the Neural Network application manager interface.

  • C: clear the cache of recently viewed images. DLTdv8 caches recently viewed images in memory so they can be re-displayed rapidly. However, sometimes during rapid frame advance in compressed files the wrong frame will be loaded and then will stay in the image cache. Using “C” clears the cache and facilitates loading the correct video image.

  • M: start multi-tracking
